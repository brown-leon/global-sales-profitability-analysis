# Global Sales and Profitability Analysis

This repository contains an applied data analysis project examining global sales transactions to understand purchasing patterns, profitability, and seasonal trends for meat and vegetable products across regions and countries.

The project focuses on data preparation (ETL), exploratory analysis, and interpretation using reproducible workflows in Python and Jupyter notebooks.

## Repository Contents

- **Global Sales and Profitability Analysis.pdf**  
  Executive project brief outlining the business context, objectives, data scope, and analytical approach.

- **Global_Sales_and_Profitability_ETL_and_Analysis.ipynb**  
  Reproducible notebook documenting data validation, cleaning, exploratory analysis, and visualisation.

- **Global_Regional_Sales_and_Profitability_Analysis_Report.pdf**  
  Analytical report presenting results and interpretation in a stakeholder-facing format.

## Notes

This project reflects student-level applied data science work and emphasises transparency, reproducibility, and disciplined analytical reasoning rather than predictive modelling or production deployment.
